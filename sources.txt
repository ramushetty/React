● What is async and defer? -    

In normal html parsing when controller sees script tag it stops parsing and fetches the script and then start parsing again 

with async attribute is present then scripts are fetched asynchronusly , parallel fetching and parsing is done 
once scripts are fetched parsing is paused and scripts are executed and then again parsing starts 


In case of defer after the html parsing is completed then scripts are executed parallelly scripts are fetched from the network
